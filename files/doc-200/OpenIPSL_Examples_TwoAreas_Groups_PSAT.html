OpenIPSL.Examples.TwoAreas.Groups.PSAT

Set of generation units modeled with PSAT

Information

This package contains four generation units, based on PSAT models, that are used in the Two_Areas_PSAT system. These models are composed only of machine models.

 G1 20kV/900MVA generation unit connected to bus 1
