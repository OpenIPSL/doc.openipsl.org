OpenIPSL.Examples.IEEE14.Generation_Groups

Set of generation unit models used in IEEE 14-bus system

Information

This package contains five generation unit models, composed of machine and exciter models, that are used in the IEEE 14-bus system. These models are:

  • GroupBus1: generation unit connected to bus 1.
  • GroupBus2: generation unit connected to bus 2.
  • GroupBus3: reactive power generation unit connected to bus 3.
  • GroupBus6: reactive power generation unit connected to bus 6.
  • GroupBus8: reactive power generation unit connected to bus 8.

Package Content

Name Description
 GroupBus1 69kV/615MVA generation unit connected to bus 1
 GroupBus2 69kV/60MVA generation unit connected to bus 2
 GroupBus3 69kV/60MVA reactive power generation unit (synchronous condenser) connected to bus 3
 GroupBus6 13.8kV/25MVA reactive power generation unit (synchronous condenser) connected to bus 6
 GroupBus8 18kV/25MVA reactive power generation unit (synchronous condenser) connected to bus 8

69kV/615MVA generation unit connected to bus 1

Information
