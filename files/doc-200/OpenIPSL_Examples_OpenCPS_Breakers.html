OpenIPSL.Examples.OpenCPS.Breakers

Models for controlled line breakers

Information

This package contains a breaker model developed especially for this example system. Since the breaker is not standard, it is not present in the main body of the library and its usage outside these examples is not encouraged.

OpenIPSL.Examples.OpenCPS.Breakers.Breaker

Controllable line breaker

Information

This breaker model is used to open and close circuits depending on a boolean gating signal named TRIGGER.

If TRIGGER is false, the breaker is open. Otherwise it is closed.
