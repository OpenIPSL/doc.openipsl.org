OpenIPSL.Examples.Tutorial.Example_3

Adding complexity and simulating a larger system

Information

This package illustrates how it is possible to model a 9-bus system and simulate it using Modelica and the OpenIPSL.

The simulation of this system also illustrates how oscillations occur on top of each other in a real system, showing that many modes can be excited with severe disturbances such as a three-phase-to-ground fault.

OpenIPSL.Examples.Tutorial.Example_3.Example_3

Fully assembled 9-bus 3-generator test system

Information

This example system represents a 9-bus 3-generator test system with a three-phase-to-ground fault at bus 9 and it must be simulated during 20 seconds.

Variables of interest are:

  • B1.v
  • B2.v
  • B3.v
  • Gen1.gen.delta
  • Gen2.gen.delta
  • Gen3.gen.delta

Note that the voltage curves present different acting modes. Also note that disturbances in the AVR reference voltage can be added to each generation unit.
