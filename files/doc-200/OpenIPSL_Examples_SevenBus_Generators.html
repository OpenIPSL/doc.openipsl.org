OpenIPSL.Examples.SevenBus.Generators

Set of generation unit models

Information

This package contains three generation unit models that are used in the SevenBus system. These models are:

  • G1: generation unit composed of machine, exciter, governor and stabilizer. Unit connected to bus FSSV.
  • G2: generation unit composed of machine, exciter, governor and stabilizer. Unit connected to bus FVALDI.
  • G3: generation unit composed of machine, exciter, governor and stabilizer. Unit connected to bus FVERGE.

OpenIPSL.Examples.SevenBus.Generators.G1

Generation unit connected to bus FSSV

Information

Generation unit number 1, connected to bus FSSV, and composed of the following component models:

  • Machine: GENROU, from PSSE.
  • Exciter: ST5B, from PSSE.
  • Turbine-Governor: IESGO, from PSSE.
  • Stabilizer: PSS2B, from PSSE.

Extends from OpenIPSL.Electrical.Essentials.pfComponent (Partial model containing all the parameters for entering power flow data).
