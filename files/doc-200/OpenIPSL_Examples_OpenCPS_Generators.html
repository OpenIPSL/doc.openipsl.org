OpenIPSL.Examples.OpenCPS.Generators

Models for the generation units used in the test systems

Information

This package contains one generation unit composed by the standard models that are available in OpenIPSL and one generation unit that has additional components representing control systems that are used to resynchronize the generation unit to the power system.

OpenIPSL.Examples.OpenCPS.Generators.G1

Generation unit connected to bus BG1

Information

24kV/100MVA generation unit connected to bus BG1, and composed of the following component models:

  • Machine: GENSAL, a salient pole synchronous machine from PSSE.
  • Exciter: SEXS, a simplified excitation system from PSSE.
  • Turbine/Governor: HYGOV, a hydro-turbine governor from PSSE.

Extends from OpenIPSL.Electrical.Essentials.pfComponent (Partial model containing all the parameters for entering power flow data).

OpenIPSL.Examples.OpenCPS.Generators.G2

Generation unit connected to bus B5

Information

24kV/100MVA generation unit connected to bus BG1, and composed of the following component models:

  • Machine: GENSAL, a salient pole synchronous machine from PSSE.
  • Exciter: SEXS, a simplified excitation system from PSSE.
  • Turbine/Governor: IEESGO, a standard turbine/governor from PSSE.

This generation unit is supposed to be disconnected in all experiments and, then, to be resynchronized to the whole system.

Extends from OpenIPSL.Electrical.Essentials.pfComponent (Partial model containing all the parameters for entering power flow data).
