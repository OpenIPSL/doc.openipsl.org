OpenIPSL.Examples.PSATSystems.ThreeArea.ThirdOrder_AVRI

ABC system with 3rd-order machines and AVR Type I

Information

This system is tests the third order machine with a type 1 AVR, from PSAT. Simulate the system for 20 seconds.

Variables of interest are:

  • order2.w
  • order3_1.w
  • order3_2.w

Note that the oscillations illustrate a clear example of non-coherent machines.

Compare these results with the ones from the other example systems within the ThreeArea subpackage.
