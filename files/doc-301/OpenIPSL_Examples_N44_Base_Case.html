OpenIPSL.Examples.N44.Base_Case.Nordic44_Base_Case

Modified version of the original Nordic 44 model

Information

This example system represents the Nordic 44 test system, represented differently from the original case, but undergoing the same line opening event.

Simulate the system for 10 seconds. If you are using Dymola, use the DAE mode. Variables of interest are:

  • bus_7020.v
  • G1_bus6500.gENSAL.SPEED

Compare these results with the ones obtained in the Nordic44_Base_Case.

Changes compared to the original case (Real data period: April, May, June and July):

  • Bus names assignation
  • New bus grouping according to the real bidding zones
  • Introduction of 5 no scalable loads at buses: 8500, 7000, 7100 and two at 6701 (to represent missing exchanges)
  • All no scalable loads have Q=P*tan(acos(0.95)) except for that one at bus 5610 with PF=0.99 (to not have frequent problems of low voltage in southern Norway)
  • Trafo 5602-5603: winding 1 ratio set from 0.9683 to 1 (to not have permanent problems of low voltage in southern Norway) and no Auto Adjust
  • Change of X about line 5400-5500 from 0.94 pu to 0.094 pu (to fix ODMS warning)
  • Branches rates (to fix some lines overloading): P_max = V_1*V_2/X_line*S_base=1*1/X_line*1000;
