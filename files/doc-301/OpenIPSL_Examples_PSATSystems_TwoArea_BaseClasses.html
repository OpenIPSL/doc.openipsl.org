OpenIPSL.Examples.PSATSystems.TwoArea.BaseClasses.BaseNetwork

Base network for TwoArea test system

Information

This base network is extended in the example systems from TwoArea subpackage. It contains a generator model connected to bus 2, represented by an order 3 machine, from PSAT.

The 31-bus system has a three-phase-to-ground fault event occurring on bus 3 at 2 seconds and lasting for 100ms.
