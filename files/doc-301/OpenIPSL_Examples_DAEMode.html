OpenIPSL.Examples.DAEMode

Test cases used to verify the efficiency of the Dymola DAE mode implementation for hybrid models

Information

This repository contains specialized versions of the Nordic44 (N44) power system model, which has been modified and used for simulation in the following paper: Erik Henningsson, Hans Olsson and Luigi Vanfretti,'DAE Solvers for Large-Scale Hybrid Models', in Proceedings of the 13th International Modelica Conference, Regensburg, Germany, March 4-6, 2019.


Note that this package has experiments that are different if compared to the original experiments.
